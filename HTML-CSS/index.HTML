Estudos

HTML e CSS

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da web.

CSS (Cascading Style Sheets) é a linguagem de estilo usada para descrever a apresentação de documentos HTML.

JavaScript

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas da web.
É amplamente utilizado para criar interatividade e dinamismo em sites.
